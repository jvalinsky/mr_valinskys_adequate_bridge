# CLI Tool

- Created `cmd/bridge-cli/main.go` using `github.com/urfave/cli/v2`.
- Implemented `account` commands:
  - `account list`: Shows all bridged accounts in the database and their active status.
  - `account add <did>`: Adds a new account to the DB (SSB feed ID derivation to be integrated fully with the manager).
  - `account remove <did>`: Deactivates an account in the DB.
- Added placeholders for `stats` and `start` commands to run the bridge engine.
- Successfully compiled to `bin/bridge-cli`.
